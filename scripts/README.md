# Blog Automation

The scripts in this directory own deterministic editorial checks and public
distribution tooling.

- Documentation scripts generate source, post, workflow, environment, social,
  video, and public-asset inventories.
- SEO scripts build discovery feeds, score rendered pages, and report content
  and internal-link opportunities.
- Social scripts validate and publish reviewed LinkedIn and Facebook plans.
- Video scripts create plans, fetch licensed assets, capture Flyto2 product
  footage, synthesize narration, render aspect-ratio variants, and run artifact
  QA.

Use [`../docs/reference/README.md`](../docs/reference/README.md) for the exact
function and file inventory. Operational commands are documented in
[`../README.md`](../README.md), [`../social/README.md`](../social/README.md), and
[`../video/README.md`](../video/README.md).
