# Project

`flyto-blog` is Flyto2's public educational content site.

Primary jobs:

- Capture SEO/AEO/GEO demand for automation, CTEM, attack surface, dark web,
  AI security, pentest, and workflow topics.
- Explain concepts in a way AI answer engines can summarize and cite.
- Route readers to landing pages for product positioning and docs for technical
  mechanics.

Content must support all five Flyto2 product lines:

- Cloud / Apps / Automation
- Security
- Data
- Zero-person Company Agent
- Big Data / Intelligence
